# QWICKY GUI Redesign — Stitch Design Reference

Generated via Google Stitch MCP. These HTML files are the design mockups for the new QWICKY GUI.

## Design System Changes (Current → New)
- **Color**: Violet (#A78BFA) → Amber/Orange (#ff9d00, #ffc485)
- **Fonts**: Outfit → Space Grotesk (headlines) + Inter (body) + Fira Code (mono/data)
- **Aesthetic**: Minimal admin → Industrial HUD / military-tactical
- **Corners**: 6px rounded → 2px sharp (almost square)
- **Layout**: Header tabs → Sidebar divisions + top section tabs
